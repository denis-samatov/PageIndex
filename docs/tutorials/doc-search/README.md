

## Примеры поиска по документам


PageIndex по умолчанию обеспечивает RAG на основе рассуждений в рамках одного документа.
Если нужно искать сразу по нескольким документам, ниже приведены три рекомендуемых сценария для разных случаев.

В исследовательской рамке многодокументный поиск распадается на два этапа: (1) выбор релевантных документов и (2) поиск по дереву внутри каждого документа. Разные сценарии ниже соответствуют разным источникам сигналов: структурированные метаданные, семантика текста или краткие описания как прокси-сигнал.

* [**Поиск по метаданным**:](metadata.md) для документов, которые можно отличить по метаданным.
* [**Поиск по семантике**:](semantics.md) для документов с разным смысловым содержанием или охватывающих разные темы.
* [**Поиск по описанию**:](description.md) легковесная стратегия для небольшого числа документов.
